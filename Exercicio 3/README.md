# Exercício 3
Implemente uma função capaz de calcular e retornar a média, o valor máximo e o valor mínimo de um array com "n" posições. 
Esta função também deve retornar uma cópia do array de entrada contendo apenas os números pares do array original, bem como o novo tamanho do array. 
Implemente essa lógica usando a linguagem C e compartilhe este código em um repositório público no GitHub.

Foi elaborado uma solução simples utilizando estrutura de dados para maior escalabiidade.
